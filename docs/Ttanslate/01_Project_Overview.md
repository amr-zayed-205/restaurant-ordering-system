# فكرة المشروع

يهدف المشروع إلى إنشاء موقع إلكتروني بسيط يتيح للعملاء تصفح قائمة الطعام الخاصة بالمطعم، واختيار الوجبات، ثم إرسال الطلب مباشرة إلى المطعم عبر تطبيق WhatsApp بطريقة منظمة وسهلة.

الهدف من المشروع هو تحسين تجربة العميل وتقليل الوقت والجهد المبذول في استقبال الطلبات يدويًا.

---

# المشكلة التي يحلها المشروع

تعتمد العديد من المطاعم الصغيرة والمتوسطة على استقبال الطلبات يدويًا من خلال WhatsApp أو المكالمات الهاتفية، مما يؤدي إلى عدة مشكلات مثل:

- كثرة الأسئلة عن الأسعار والأصناف.
- إرسال المنيو على هيئة صور أو ملفات PDF يصعب تصفحها.
- تكرار طلب بيانات العميل في كل مرة.
- استهلاك وقت كبير من موظف خدمة العملاء.
- احتمال فقدان بعض الطلبات أو حدوث أخطاء أثناء تسجيلها.

---

# الحل المقترح

يوفر النظام موقعًا إلكترونيًا يسمح للعميل بـ:

- تصفح قائمة الطعام.
- مشاهدة صور الوجبات وأسعارها.
- البحث داخل المنيو.
- إضافة الوجبات إلى سلة الطلبات.
- إدخال بياناته مرة واحدة.
- إرسال الطلب كاملًا إلى المطعم عبر WhatsApp.

---

# أهداف المشروع

يسعى المشروع إلى تحقيق الأهداف التالية:

- تحسين تجربة طلب الطعام.
- تقليل الوقت المستغرق في التواصل بين العميل والمطعم.
- تقليل الضغط على خدمة العملاء.
- توفير تجربة استخدام بسيطة وسريعة.
- تدريب الفريق على العمل الجماعي باستخدام بيئة تطوير احترافية.
- إنتاج مشروع يمكن إضافته إلى معرض الأعمال (Portfolio).

---

# المستخدم المستهدف

المستخدم الأساسي للنظام هو:

**العميل (Customer)**

وسيكون قادرًا على:

- تصفح المنيو.
- البحث عن الوجبات.
- مشاهدة تفاصيل كل وجبة.
- إضافة المنتجات إلى السلة.
- إتمام عملية الطلب.
- إرسال الطلب إلى المطعم.

# نطاق المشروع (Project Scope)

## النسخة الأولى (Version 1)

- تصميم واجهة المستخدم.
- عرض المنيو.
- عرض تفاصيل الوجبات.
- استخدام بيانات تجريبية (JSON).

---

## النسخة الثانية (Version 2)

- إنشاء Backend باستخدام ASP.NET Core Web API.
- إنشاء قاعدة البيانات.
- ربط React بالـ API.
- عرض البيانات القادمة من قاعدة البيانات.

---

## النسخة الثالثة (Version 3)

- إنشاء سلة المشتريات.
- صفحة إتمام الطلب.
- إدخال بيانات العميل.
- إرسال الطلب عبر WhatsApp.

---

## النسخة الرابعة (اختيارية)

إضافة مميزات تعتمد على الذكاء الاصطناعي مثل:

- اقتراح وجبات مشابهة.
- اقتراح وجبات حسب الميزانية.
- البحث الذكي.
- عرض أكثر الوجبات طلبًا.

- العمل الجماعي.
- إدارة المشروع باستخدام GitHub Projects.
